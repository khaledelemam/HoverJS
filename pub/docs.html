HoverJS

Documentation

Getting started

Add the following code to the head in your HTML file 

  
    <link rel="stylesheet" type="text/css" href="HoverJS.css">
    <script defer type="text/javascript" src='js/HoverJS.js'>
To use the library: 

  
    const hv = new Hover()

API

setFrame(id, width:optional, height:optional)
sets up hover preview with the save button when mouse is over the element. Takes in element id (must), height and width of the preview frame. Dimensions are optional, the default is 400 by 400px.
Cursor has to be on element for at least 0.5 seconds for frame to show.
createList(id:optional)
creates draggable pop-up list. If input is element id, it creates a checkbox of element with hyperlink by calling createCheckbox
createCheckbox(id)
creates checkbox corresponding to the element which its id is the input.
removeCheckbox(id)
removes checkbox of element.
showList()
shows pop-up list.
hideList()
hides pop up list.
getTotalItems()
return total number of items in the pop-up list.
getItemsList()
returns list of elements in the pop-up list
listBackgroundColor(color)
takes in hexadecimal color code and changes the background color of the list.
